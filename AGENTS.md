# AGENTS.md

## Scopo

Questa repository contiene il catalogo ufficiale delle skill Codex.

## Naming

Tutte le skill devono:

- essere minuscole
- usare kebab-case
- non usare underscore
- non usare spazi
- non usare accenti

Esempio:

as-common-codex-command-pack

## Struttura minima

nome-skill/

    SKILL.md

## Struttura consigliata

nome-skill/

    SKILL.md
    references/
    examples/
    scripts/

## Sicurezza

Non salvare:

- password
- token
- api key
- certificati
- file .env

## Workflow

1. creare skill
2. validare skill
3. aggiornare indice
4. commit
5. push
