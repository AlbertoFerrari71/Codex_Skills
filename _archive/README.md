# Archive

Questa cartella contiene backup storici mantenuti temporaneamente per tracciabilita.

I file qui presenti:

- non sono skill attive;
- non devono essere considerati dal validator come skill operative;
- non devono essere usati come fonte primaria;
- possono essere rimossi in uno step dedicato dopo verifica manuale.

I backup delle skill sono raccolti in:

```text
_archive/backup-skills/
```
