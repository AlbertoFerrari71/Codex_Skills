# Scopo

Usa questa skill per progetti Python, FastAPI, pytest, uvicorn e debug incrementale nei progetti di Alberto.

# Quando usarla

Usala per bug Python/FastAPI, test pytest, virtualenv, endpoint, template, route, configurazione ambiente, log e verifiche incrementali.

# Procedura

1. Preferire modifiche piccole e verificabili.
2. Controllare struttura progetto e file coinvolti.
3. Non introdurre nuove dipendenze senza motivo.
4. Usare test esistenti prima di crearne di nuovi.
5. Separare errore osservato, causa probabile e fix proposto.
6. Dopo ogni modifica proporre comandi di verifica semplici.
7. Mantenere codice leggibile e minimale.

# Output atteso

- Diagnosi breve.
- File da modificare.
- Modifica proposta.
- Comandi di test.
- Rischi residui.
- Prossimo passo.
