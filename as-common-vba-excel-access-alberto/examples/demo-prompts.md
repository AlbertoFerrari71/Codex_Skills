# Demo Prompts

## Excel VBA

```text
Prepara una macro Excel VBA per importare un CSV in un foglio dedicato.
Usa separatore formule italiano, codice semplice, flag debug se utile.
Non creare nuovi fogli senza spiegare nome e posizione.
Indica dove incollare il codice e come testarlo.
```

## Access Read-Only

```text
Prepara una query Access/ADODB read-only verso eSolver.
Usa placeholder per i campi credenziali.
Se le relazioni non sono fisiche, documenta le join ipotizzate.
Restituisci codice, riferimenti necessari e test manuale.
```
